Les systèmes alternatifs

Haiku

Haiku, anciennement OpenBeOS, est un système d'exploitation crée en 2001 afin d'offrir une version open-source de BeOS au moment où ce dernier était abandonné avec le rachat de Be Inc. par Palm Inc. Haiku n'est donc pas basé Linux ou BSD, mais sur BeOS. L'environnement graphique est le même que BeOS, Tracker. Haiku utilise son propre système de système de gestion de paquets, hpkg. Haiku est capable de faire fonctionner des programmes pour BeOS dans sa version 32 bit ; mais aussi les programmes Linux et Windows grâce à Wine.

Haiku est parmi les systèmes alternatifs, celui qui est le plus utilisable dans une utilisation quotidienne. Il est capable de faire tourner LibreOffice et supporte des navigateurs modernes comme Falkon ou GNOME Web.

Site web : www.haiku-os.org
